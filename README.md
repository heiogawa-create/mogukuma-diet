# もぐクマDiet

AIダイエットコーチキャラクター「もぐクマ」と一緒に食事・体重をゆるく記録するスマホファーストMVPです。

## 技術構成

- Next.js
- TypeScript
- Tailwind CSS
- React
- Recharts

## MVP機能

- ホーム画面
- 食事写真アップロード画面
- 食事記録フォーム（食事名、朝食/昼食/夕食/間食、カロリー、PFC、食物繊維、水分量）
- 保存した食事記録の一覧表示、編集、削除
- 今日の合計カロリー、PFC、食物繊維、水分量の自動計算
- 体重記録フォーム（今日の体重、目標体重）
- 保存した体重を反映するグラフ表示
- localStorage保存によるページ更新後の記録保持
- もぐクマのAIコメント表示

OpenAI APIキー未設定でも、手入力とlocalStorage保存で動作確認できます。

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開いてください。
