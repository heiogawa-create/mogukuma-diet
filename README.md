# もぐクマDiet

AIダイエットコーチキャラクター「もぐクマ」と一緒に食事・体重をゆるく記録するスマホファーストMVPです。

## 技術構成

- Next.js
- TypeScript
- Tailwind CSS
- React
- Recharts

## MVP機能

- ホーム画面
- 食事写真アップロード画面
- ダミーのAI食事解析結果表示
- カロリー/PFC表示
- 体重記録画面
- グラフ表示
- AIコメント表示

OpenAI APIキー未設定でも、ダミーデータで動作確認できます。

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開いてください。
